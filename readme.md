# 三维点云处理：算法与实战

本课程是3D视觉工坊的课程，计算机视觉的最终体现是三维视觉，而三维视觉的表达方式则是点云，点云处理在整个三维视觉领域占有非常重要的地位，几乎涉及到所有相关领域，例如自动驾驶感知定位、SLAM、三维场景重建、AR/VR、SFM、姿态估计、三维识别、结构光、立体视觉、三维测量、视觉引导等。
